Sistema de ventas

Este sistema, es el tipico sistema de control de ventas. La pretensión de este sistema se describe en los siguientes puntos:
1. Trabajar su desarrollo en dos diferentes equipos, sin tener que transportar el proyecto en memoria
2. Permitir buscar articulos por captura de voz
3. Permitir reconocer articulos por reconocimiento de imagenes
